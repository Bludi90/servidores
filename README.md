# Servidores — Documentación (solo-Docs)
Repositorio privado para **documentación y estado** de los servidores activos.

## Contenido
- `docs/`: guías, decisiones, comandos personalizados.
- `state/`: “fotografías” horarias del estado (discos, docker, VMs, backups) **sanitizadas**.
- `scripts/`: automatización (recopilar estado, commit & push).

## Política de seguridad
- **Nunca** subir claves, contraseñas, tokens ni confs reales.
- IPs y rutas sensibles se **anonimizan** en `state/`.
- Este repo es de solo documentación; configuraciones reales viven fuera.
